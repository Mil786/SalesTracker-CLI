# SalesTracker CLI 🚀

Sistema de rastreamento de vendas desenvolvido em **C#** e **.NET 10** como projeto final do Módulo 1 (Lógica de Programação Backend).

## 📝 Sobre o Projeto
O SalesTracker é uma aplicação de linha de comando (CLI) que permite aos usuários registrar vendas, calcular lucros individuais e visualizar um balanço financeiro final. O foco deste projeto foi consolidar conhecimentos em estruturas de dados simples e lógica de fluxo.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** C#
- **Plataforma:** .NET 10
- **IDE:** VS Code

## 🧠 Conceitos Aplicados
- **Tipagem Estrita:** Uso de `decimal` para precisão financeira.
- **Coleções:** Manipulação de múltiplas `List<T>`.
- **Estruturas de Repetição:** Utilização de `for` para entrada e saída de dados.
- **Formatação de String:** Uso de interpolação, `PadRight` e máscaras numéricas (`:N2`).
- **Lógica Condicional:** Sistema de tomada de decisão para status de lucro/prejuízo.
